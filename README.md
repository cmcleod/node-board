Where in the World
===========================

Setting up
----------

### Install Node and NPM

See [Node.js.org](http://nodejs.org/download/) for details.

Make sure */usr/local/share/npm/bin* is in your *$PATH*

### Set up Grunt

	# While in your git repo
	sudo npm install -g grunt-cli;
	npm install;
